# シフトカレンダー 技術仕様書

## システム概要

シフト管理Excelファイルを解析し、遅番・休み・土曜出勤の情報をJSON形式に変換してカレンダー表示するWebアプリケーション。GitHub Pagesで静的にホスティングし、スマホブラウザから閲覧する。

## アーキテクチャ

```
Excel (.xlsx)
  |
  v
convert.py (Python)
  |
  v
JSON (data/*.json)
  |
  v
GitHub Pages (静的ホスティング)
  |
  v
スマホブラウザ (index.html)
```

## ファイル構成

```
シフトカレンダー/
  index.html          ... カレンダー表示画面（HTML/CSS/JS一体型）
  convert.py          ... Excel→JSON変換スクリプト
  data/
    months.json       ... 月一覧インデックス（キー: "YYYY-MM", 値: "YYYY年M月"）
    2026-04.json      ... 月別シフトデータ
    ...
  docs/
    手順書.md
    仕様書.md
```

## Excelファイルフォーマット

対象ファイル名: `シフト管理YYYY年MM月度.xlsx`

最初のシート（シート名不問）を読み込む。

| 行 | A列 | B列〜 |
|----|------|-------|
| 1行目 | 年月度（例: 「2026年04月度」） | 日付（1, 2, 3, ...） |
| 2行目 | （空） | 曜日（月, 火, ...） |
| 3行目 | （行事予定ヘッダ） | 行事予定テキスト |
| 4行目〜 | スタッフ氏名（姓 名） | シフトパターン |

### スタッフ名の処理

- 姓のみ表示する（例: 「田中 太郎」→「田中」）
- 同姓のスタッフがいる場合は名の頭文字で区別する（例: 「田中(太)」「田中(花)」）

### 除外行

A列が以下に該当する行はスタッフとして扱わない:
- 「行事予定」「打刻端末」
- 日付形式（例: 「4/1」）で始まる行

## convert.py の処理内容

### 入力

- コマンドライン引数でExcelファイルパスを指定
- 引数なしの場合、スクリプトと同じフォルダおよびユーザーのダウンロードフォルダから `シフト管理*.xlsx` を自動検索

### 処理

1. A1セルから年月を正規表現で抽出する
2. 1行目から日付（1〜31の数値）とその列位置を取得する
3. 2行目から曜日、3行目から行事予定を取得する
4. 4行目以降のスタッフ名とシフトデータを取得する
5. 同姓判定を行い表示名を決定する
6. 月初日の曜日（Mon=0〜Sun=6）と月の日数を算出する

### 出力

`data/` フォルダに以下を出力する:

**月別JSON（例: `2026-04.json`）**

```json
{
  "year": 2026,
  "month": 4,
  "daysInMonth": 30,
  "startDow": 2,
  "days": [1, 2, 3, ...],
  "dow": ["水", "木", "金", ...],
  "events": ["", "入社式", "", ...],
  "staff": [
    {
      "name": "田中",
      "shifts": {
        "1": "早番",
        "2": "遅番 / 備考",
        ...
      }
    }
  ]
}
```

**月一覧インデックス（`months.json`）**

```json
{
  "2026-04": "2026年4月",
  "2026-05": "2026年5月"
}
```

既存の `months.json` がある場合はマージして上書きする。

## index.html の仕様

### カレンダー表示

- 月曜〜土曜の6列グリッドで表示する（日曜は非表示）
- 月初日の曜日に応じて先頭に空セルを配置する
- 土曜列は背景色を薄い青にする

### 表示対象のシフトパターン

シフトセルの値をスラッシュ区切りの先頭部分（パターン）で判定する。

| 種別 | 判定条件 | タグ色 |
|------|----------|--------|
| 遅番・中番 | パターンに「遅」を含む、または「中1F」「中2F」で始まる | オレンジ（背景: #fff3e0, 文字: #e65100） |
| 休み・有給 | パターンに「休」を含む、またはパターンが「有給」と一致 | ピンク（背景: #fce4ec, 文字: #c62828） |
| 土曜出勤 | パターンが「土」で始まる | 紫（背景: #f3e5f5, 文字: #6a1b9a） |

### 行事予定

日付の下に行事予定テキストを全文表示する（オレンジ背景のラベル）。

### データ読み込み

- 起動時に `data/months.json` をfetchして月一覧を取得する
- プルダウンで月を選択すると `data/YYYY-MM.json` をfetchしてカレンダーを描画する
- 取得したデータはメモリにキャッシュし、同じ月の再選択時はfetchしない
- デフォルトで最新月を表示する

### レスポンシブ対応

- 768px以上の画面ではフォントサイズやセルの高さを拡大する
- スマホ表示を主な利用環境として想定している
