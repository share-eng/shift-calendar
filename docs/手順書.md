# シフトカレンダー 操作手順書

## 概要

シフト管理Excelファイルから遅番・休み・土曜出勤の情報を抽出し、カレンダー形式で表示するWebアプリケーション。
スマホからいつでもシフトを確認できる。

公開URL: https://share-eng.github.io/shift-calendar/

## 必要なもの

- Python 3（コマンドプロンプトで `python --version` で確認）
- openpyxlライブラリ（Excelファイル読み込み用）
- Git（データのアップロード用）

## 新しい月のExcelを追加する手順

### 1. Excelファイルを準備する

シフト管理Excelファイル（例: `シフト管理2026年05月度.xlsx`）をダウンロードまたはコピーする。

### 2. コマンドプロンプトを開いてフォルダに移動する

```
cd cowork\シフトカレンダー
```

### 3. 変換スクリプトを実行する

```
python convert.py "シフト管理2026年05月度.xlsx"
```

ダウンロードフォルダにあるファイルを指定する場合:

```
python convert.py "%USERPROFILE%\Downloads\シフト管理2026年05月度.xlsx"
```

引数なしで実行すると、同じフォルダとダウンロードフォルダ内の「シフト管理*.xlsx」を全て変換する:

```
python convert.py
```

「完了！ 1件変換しました。」と表示されれば成功。

### 4. GitHubにアップロードする

```
git add data/
git commit -m "5月追加"
git push
```

### 5. 反映を確認する

1分ほどで https://share-eng.github.io/shift-calendar/ に反映される。

## スマホでの閲覧方法

1. スマホのブラウザで https://share-eng.github.io/shift-calendar/ にアクセスする
2. 画面上部のプルダウンで表示する月を切り替える
3. カレンダー上で遅番（オレンジ）、休み（ピンク）、土曜出勤（紫）を確認できる

## トラブルシューティング

### 「python が見つかりません」と表示される

Pythonがインストールされていない。以下からインストールする:
https://www.python.org/downloads/

インストール時に「Add Python to PATH」にチェックを入れること。

### 「No module named 'openpyxl'」と表示される

openpyxlライブラリがインストールされていない。以下のコマンドでインストールする:

```
pip install openpyxl
```

### 「年月を読み取れません」と表示される

Excelファイルのシート名またはA1セルの形式が想定と異なっている。
A1セルに「2026年05月度」のような年月表記があることを確認する。

### pushしても反映されない

- GitHub Pagesの反映には1〜2分かかる場合がある
- ブラウザのキャッシュをクリアして再読み込みする
- `git status` でpushし忘れがないか確認する
